Course Description

This course is designed to provide students with an in-depth learning experience around database modeling, design, and implementation. It is divided into two major modules. In the first module, Database Design, students learn to analyze complex business scenarios and create a data model, a conceptual representation of an organizationâ€™s information. In the second module, Database Programming with SQL, students implement their database design by creating a physical database using the Oracle Application Express tool and SQL, the industry-standard database programming language. Students will gain extensive hands-on experience with databases. Upon completion of this course, students will have been prepared to sit for the Oracle Database SQL Expert exam at an Oracle-authorized testing center (if they wish to pursue certification.) This will require the students to pay a fee and schedule the test.
